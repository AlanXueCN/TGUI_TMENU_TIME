
TGUI_TMENU_TIME 由以下几大模块构成： 
  TGUI：(Text Graphical User Interface)  一个“轻量级的，基于文本显示缓冲区的准GUI系统”，
主要用于单片机驱动的，单色点阵显示屏应用中。对文本显示方式的支持。 
  TMENU：(Text Menu) 以 TGUI 为基础实现的完整菜单系统解决方案，实现了对各种参数的调
整支持，如：模式切换，数值调整等。 
  TIME：(Text IME) 以TGUI为基础，实现了数字键盘方式的输入法支持，操作方法与按键手机
类拟。 
因TGUI/TMENU/TIME均以TGUI为基础延伸出的抽像应用模块。故在此手册中， 无特殊说明， “TGUI
系统”特指“TGUI、TMENU、TIME”模块组成的系统。而TGUI仅指 TGUI。 
TGUI 系统以 C 语言源代码方式提供，在源代码中的所有结构与函数均以“T”打头以便与用户代码
进行区分。 
TGUI 系统致力于满足在不同编译环境，不同单片机上的使用。

TGUI_TMENU_TIME源代码及项目演示: http://download.csdn.net/detail/thtfcccj/5995059
 
应用情况概要(截止至2012年)
从2009年初开发完成至今，经实际使用到了不同公司的超过 10个产品中(仅粗略统计)，多种场合的
应用，直接或间接地促进了 TGUI系统的发展与更新： 

  2009.7~,工业用臭氧发生器控制部分(自已)：TGUI做主界面显示，TMENU 调整参 
  开始TGUI，TMENU 系统的开发设计并第一次使用 
  使用Arm内核 LPC2103+ST7920(64X32)+IAR编译器 
  实现了对显示屏接口的标准化。 
  ST7920 显示屏驱动的支持，使用 ST7920 自带字库。 

  2009.10~,光纤感温控制器(朋友)：主要用作主界面显示，配合Tmenu菜单系统调整参数。 
  使用C51内核uPSD3354+ST7920(192X64)+KeilV3编译器 
  增加了对多主界面的支持 
  增加了对固定显示窗口的支持(最上一行显示运行状态) 
  加入了以ID 号供替结构体内函数指针的功能，以实现对C51 Keil编译器的支持。 
  加入了对浮点数数值调整的支持； 
  Tmenu增加了对查找表的支持，以实现根据权限或应有不同显示不同的菜单或项目。 
  对多国语言显示支持的验证，实现了直接通过菜单，对中文，英文界面的直接切换。 
  对菜单系统直接强制跳转功能的支持。 

  2009.10~,喷烤漆控制器(同事)：主要用作主界面显示，配合Tmenu菜单系统调整参数。 
  用AVR内核Mega32+ST7920(64X32)+GCC编译器(用AVR Studio 4) 
  增加以ID 号供替常量结构体内的指针，以实现对AVR 单片机应用的支持。 
  TMENU加入了对极少按键功能的支持，配置好后，最少仅需“确定，上，下”三个键即可
查看修改所有参数。 

  2010.2(约),某供热采集控制装置(朋友)：主要用作主界面显示，配合Tmenu 菜单系统调整参数。 
  用MSP430内核+ T6963+ IAR 编译器 
  增加了对T6963 显示屏驱动的支持 
  增加对自定义字模(定制字母和定制汉字)的支持 
  验证了对MSP430 内核的支持。 

  2010.4 可燃气体控制器(自已+同事)，主要用作主界面显示，配合Tmenu 菜单系统调整参数 
  用ARM内核+T6963(240*128)+ IAR 编译器 
  增加了对全定制汉字库的支持(字模存储在外部Spi Flash中) 
  配合TGUI的多窗口支持，实现了在主界面上的弹出式窗口；   
  TMenu增加动态显示样式支持。显示样式(边框，滚动条等)可根据环境不同运态改变。 
  TMenu 增加了动态菜单(通过一个 TMenu_t 结构体内部内容的改变，实现各种层次的菜单)
支持。用于将原应用中的“抽像数据接口” 实现在 TGUI中以完成用户的交互(原使用UCGUI
实现)。 

  2011.x 可燃气体控制器(自已)二期开发项目中。增加了TIME模块用于对文本输入法的支持： 
  TIME以 TGUI为基础，通过使用与带数据键手机类拟的键盘(0-9,*,#)，实现了： 
  符号输入法， 
  数字输入法， 
  大写字母输入法， 
  小写字母输入法， 
  拼音输入法功能， 
  输入法间的切换与管理。 

  2011至今：多款供热系统控制装置或其它设备(朋友)：具体使用情况不详(更新略)。 

  2011.4~ 混凝土氯离子测量（自已） ，主要用作主界面显示，配合Tmenu 菜单系统调整参数 
  用AVR内核+ KS0108+ IAR 编译器 
  增加了对KS0108（128*64）驱动支持 
  自定义字模部分增加了对不同取模方式的支持 

  2012.4~ 混凝土扩散系数测量（自已），主要用作主界面显示，配合Tmenu 菜单系统调整参数 
  用Arm C0内核+T6963（192*64）+ IAR 编译器 
  完善增加了说明部分 

  2012.7~ 混凝土电通量测量（自已），主要用作主界面显示，配合Tmenu 菜单系统调整参数 
  用Arm C0内核+T6963（192*64）+ IAR 编译器 
  应用已基本保持稳定2.2  由产品引起的软件更新
